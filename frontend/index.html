Safe Local Malware Analysis (Synthetic Only)

Safety Checklist

  • Use a VM or snapshot; keep offline.
  • Only use bundled samples under samples/.
  • Execution runs in a worker with no network.
  • Timeouts and resource caps are enforced per run.
  • Any networking is to 127.0.0.1 only, inside the worker.

Execution buttons remain disabled until confirmed.
